CONTACT WEBSITE

Web Design

For my GrC 218 Typography class we created a contact page using HTML and CSS through brackets. Our websites included links to our resume, linkedin and more.

What I loved most about the project was being able to include our personal identity and creating a web page that we felt represented part of ourselves. I loved that we could make it our own and had freedom to do what we wanted with some guidelines. What I enjoyed most was being able to use HTML and CSS and directly see what I was creating.
